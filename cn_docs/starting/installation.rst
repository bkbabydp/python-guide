正确安装Python
================

很有可能你的操作系统上已安装了Python。

如果是的话，那么在使用Python之前就不用安装或者配置任何其他东西了。
虽说如此，我还是强烈推荐你在构建实际用途的Python应用程序之前，安装本指南之后会介绍的工具和第三方库。特别是，你应该安装Distribute，Pip，以及Virtualenv---它们能助你更加简便地其他第三方Python库。
